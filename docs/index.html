Simple Canvas Documentation

Simple Canvas is a Python library that enables instructors to quickly develop online course materials from a set of plain text files. The Simple Canvas command line interface simplifies the organization of modules, the creation of course materials from templates, and their upload to the Canvas learning management system (Canvas LMS by Instructure) through the Canvas API.

Follow the tutorial below for a basic introduction to Simple Canvas. Refer to the “How-to” guides for more advanced customization.
